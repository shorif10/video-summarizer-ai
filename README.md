# 🎬 Video Summarizer AI

AI-powered YouTube video summarizer using Google Gemini API.

## ✨ Features
- YouTube video transcript extraction
- AI-powered summarization (Gemini)
- Key points extraction
- Summary history

## 🛠️ Tech Stack
- FastAPI (Backend)
- Google Gemini API (LLM)
- Streamlit (Frontend)
- SQLite (Database)

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Gemini API Key (free at ai.google.dev)

### Installation
```bash
git clone https://github.com/username/video-summarizer-ai
cd video-summarizer-ai
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# Add your API key in .env
```

### Run
```bash
uvicorn src.main:app --reload
```
